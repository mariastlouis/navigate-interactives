DATA MAP: Presidential voting trends

About this map

This map was originally made for The (Colorado Springs) Gazette. It combines geometry boundary files of the precincts with voter registration data and precinct-level voting results to show which presidential candidate won in each precinct.

Findings: Democratic supporters are more likely to live in the downtown core of Colorado Springs and strong Republican supporters are more likely to live on the eastern plains of El Paso County.

Options for similar maps: This map is unique because of the geometry -- the individual precinct geometry boundary files that had to be requested from El Paso County. If there is a geometry boundary file out there, then there is a map of information just waiting to be made.
