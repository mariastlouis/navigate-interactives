DATA MAP: School district taxes

About this map

To make this map, we had to calculate tax rates for individual districts in the Pikes Peak region and give our self-created dataset a key to join it with an edited geometry boundary file containing just the regional school districts.

Findings: Tax rates for school districts in the Pikes Peak region vary greatly -- from $394 on a $200,000 home in Fountain, Colo. to $959 for a home with the same value in northern Colorado Springs.

Options for similar maps: This map is special because of our ability to calculate the data and edit GIS shapefiles to fit it. If there is data out there, we can make a map out of it, even if it's never been done before.
